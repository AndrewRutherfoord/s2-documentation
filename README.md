## Project structure
- `structured-documentation/` contains the "single source of truth" for the S2 documentation
- `structured-documentation-generator/` contains the code that does initial generation of the documentation in `structured-documentation/`
- `website/` contains the Docusaurus-based S2 documentation website
- `website-generator/` contains the code that generates the documentation in `website/docs/API/` from the structured documentation
